#Vire Studios

Vire Studios is a small independent game development project focused on building FPS games.

It’s an early stage startup led by KazuDev with a clear goal: build strong foundations, create responsive gameplay, and develop competitive shooter experiences that feel good to play.

Right now the work consists of prototypes, core gameplay systems, and backend infrastructure. Everything is being built step by step with long term growth in mind.

The goal is simple: create FPS games that are competitive, polished, and built with intention from the start.
